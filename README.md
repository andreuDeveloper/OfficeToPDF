# OfficeToPDF
This library allows to Transform an Office document or other documents to a final PDF.

Common extensions:
  - ".doc"
  - ".docx"
  - ".txt"
  - ".rtf"
  - ".html"
  - ".htm"
  - ".xml"
  - ".odt"
  - ".wps"
  - ".wpd"
  - ".css"
  - ".json"
  - ".xls"
  - ".xlsb"
  - ".xlsx"
  - ".ods"
  - ".csv"
  - ".ppt"
  - ".pptx"
  - ".odp"
    
Can transform other formats like: .csv, .json..
